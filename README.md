## 🛠️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/alireza-abarian786/first-react-project.git
   ```
2. **Navigate to the project**
   ```bash
   cd first-react-project
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Run the project**
   ```bash
   npm start
   ```
